About Tetris

Tetris is a classic puzzle game where players must fit falling blocks together to clear lines and score points. The game ends when the blocks stack up to the top of the screen.

How to Play

  • Use the arrow keys to move and rotate the falling blocks.
  • Complete horizontal lines to clear them from the screen.
  • The game speeds up as you clear more lines, so stay alert!

Game Controls

  • Left Arrow: Move block left
  • Right Arrow: Move block right
  • Up Arrow: Rotate block
  • Down Arrow: Speed up block descent
